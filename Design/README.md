# Desing Folder

This directory contains the Syetem Design and Technical Documentation, including system design decisions and developer guides for the Climate-Driven Energy Demand Analytics System.

Developers should refer to these documents during the implementation phase to ensure alignment with the established architecture and coding standards.

## Core Documentation
*   [DESIGN.md](DESIGN.md): Detailed system design, including component interactions and data flow.
*   [PRODUCT.md](PRODUCT.md): User-focused product definition, goals, and features.

## Developer How-To Guides
*   [AUTO-FORMATTING.md](AUTO-FORMATTING.md): Instructions on how to use automated tools to ensure code quality, linting, and formatting.
*   [HOW-TO_TRAEFIK.md](HOW-TO_TRAEFIK.md): Guide for configuring and managing Traefik as the reverse proxy for the system.
*   [STYLE_GUIDE.md](STYLE_GUIDE.md): Guide for frontend components.
